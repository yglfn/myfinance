Panduan Setup Google Sheets
  1. Buka Google Sheets dan buat spreadsheet baru
  2. Klik Extensions > Apps Script
  3. Hapus semua kode default, paste isi file Code.gs
  4. Klik Save (ikon floppy disk), lalu Deploy > New deployment
  5. Klik ikon roda gigi di samping "Select type", pilih Web app
  6. Description: My, Execute as: Me, Access: Anyone
  7. Klik Deploy, izinkan akses jika diminta
  8. Salin URL yang muncul, paste di atas
Penting: Jika sudah pernah deploy, gunakan New deployment (bukan Manage), karena update lama tidak mengubah URL
